# Life-Purse-AI
Smart Cardiac Analysis System
Developed an AI-powered cardiac assessment system that predicts Left Ventricular Ejection Fraction (LVEF) from standard 12-lead ECG images using a MobileNetV2 deep learning model. Implemented image preprocessing, transfer learning, hyperparameter optimization with Optuna, and model training using the Adam optimizer. Achieved an MAE of 4.94 and MSE of 36.17 for EF estimation. Integrated the trained model into a Flutter-based mobile application using TensorFlow Lite, enabling healthcare professionals to upload ECG images and receive instant EF predictions as a clinical decision-support tool.

Technologies

Python, TensorFlow, Keras, MobileNetV2, Optuna, TensorFlow Lite, Flutter, Deep Learning, Computer Vision, Medical AI
